# Daily Planner

## Overview
This a Typescript based daily planner.

## Tech Stack
  - Front-End: Typescript

## User Stories
  - User can create a task.
  - User can read a task.
  - User can update a task.
  - User can delete a task.

## Project In Action
- User can perform all CRUD operations<br />
